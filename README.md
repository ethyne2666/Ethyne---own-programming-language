# Ethyne Programming Language

Ethyne is a simple JavaScript-inspired programming language built from scratch using Node.js.

The goal of this project is to understand how programming languages work internally by implementing every major component of a language, including lexical analysis, parsing, Abstract Syntax Trees (AST), interpretation, code generation, error handling, and command-line tooling.

This project was built as an educational compiler/interpreter project and serves as a foundation for future language development.

---

* Custom `.eth` source files

* Parser

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/ethyne.git
```

Move into the project directory:

```bash
cd ethyne
```

Install dependencies:

```bash
npm install
```

---

## CLI Commands

### Run a Program

Execute an Ethyne source file directly:

```bash
node cli.js run examples/hello.eth
```

or

```bash
ethyne run examples/hello.eth
```

---

### Build (Transpile to JavaScript)

Convert Ethyne code into JavaScript:

```bash
ethyne build file.eth
```

---

### Build to a File

Generate JavaScript and save it:

```bash
ethyne build file.eth -o output.js
```

---

### Display Tokens

Print the token stream generated by the lexer:

```bash
ethyne tokens file.eth
```

---

### Display AST

Print the Abstract Syntax Tree:

```bash
ethyne ast file.eth
```

## How Ethyne Works

The language processing pipeline follows these steps:

```text
Source Code
     в”‚
     в–ј
 Lexer (Tokenizer)
     в”‚
     в–ј
 Parser
     в”‚
     в–ј
 Abstract Syntax Tree (AST)
     в”‚
     в”њв”Ђв”Ђв–є Interpreter
     в”‚
     в””в”Ђв”Ђв–є JavaScript Code Generator
```

### Lexer

The lexer converts source code into tokens.

Example:

```eth
cheppu("Hello");
```

becomes:

```text
IDENTIFIER  cheppu
LPAREN      (
STRING      Hello
RPAREN      )
SEMICOLON   ;
```

---

### Parser

The parser converts tokens into an Abstract Syntax Tree (AST).

Example:

```json
{
  "type": "PrintStatement",
  "value": "Hello"
}
```

---

### Interpreter

The interpreter walks through the AST and executes the program directly.

---

### Code Generator

The code generator converts Ethyne AST nodes into JavaScript code.

This allows Ethyne programs to be transpiled and executed through Node.js.
