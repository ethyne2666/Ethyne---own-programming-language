# Contributing

Thank you for your interest in contributing to Ethyne.

## Development Setup

```bash
git clone https://github.com/YOUR_USERNAME/ethyne.git
cd ethyne
npm install
```

## Branch Naming

```text
feature/new-feature
bugfix/fix-parser
```

## Commit Messages

```text
feat: add arrays
fix: parser bug
docs: update readme
```

## Pull Requests

Please:

- Keep changes focused
- Add examples if necessary
- Update documentation
- Ensure existing features continue working
